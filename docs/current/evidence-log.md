## Documentation drift review

| Claimed capability | Evidence source | Observed release state | Classification | Required action |
| --- | --- | --- | --- | --- |
| Markdown, HTML, and JSON documentation representations | Existing docs trees | Present for portions of the release, but coverage and indexing were inconsistent across guide families | Drift | Generate `docs/current` counterparts and validate them as a contract |
| Public documentation portal | Existing `docs/index.html` | Present, but not consistently sourced from a current canonical guide set | Drift | Generate a current portal and map from canonical Markdown |

## Validation record

- `pnpm docs:check` validates counterpart coverage, metadata, checksums, JSON parsing, local links, and portal inclusion.

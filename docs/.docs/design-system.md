# GIDEVO API Tool Design System

## Tone
Neural / deep-space CLI identity

## Mode
`dark`

## Core tokens
- Background: `#0b1220`
- Panel: `#111827`
- Text: `#e5e7eb`
- Muted: `#94a3b8`
- Line: `#334155`
- Accent: `#7c3aed`
- Accent 2: `#2563eb`
- Success / Warning / Danger: `#059669` / `#d97706` / `#dc2626`
- Radius: `12px`
- Font: `Arial, Helvetica, sans-serif`
- Mono: `Consolas, monospace`

## Docs integration
- Shared CSS: [design-system.css](./design-system.css)
- Machine tokens: [design-system.tokens.json](./design-system.tokens.json)
- All ultra-linked HTML pages are rendered with this theme (not a generic shared skin).

## Operator UX standards
- Semantic HTML and visible focus states
- High-contrast product palette (no generic purple-gradient AI slop)
- Prefer product density and operational clarity
- Respect `prefers-reduced-motion`
